Design Document: Network Monitoring and Filter Tool

Files and Their Purposes
-----------------------

network_monitor.py
  - The main Python script that runs the program.
  - Loads the blacklist, monitors network connections, checks for blacklisted IPs, logs violations, and displays status in the console.

blacklist.txt
  - A text file listing IPs to monitor against.
  - Each line contains one IP address (e.g., 8.8.8.8).

violations.log
  - A log file generated at runtime.
  - Stores records of blacklisted IP connections with timestamps.
